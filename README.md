# TalentLens AI

### Hire the Best Future Employee

TalentLens AI is an AI-powered candidate intelligence and shortlisting platform built to move beyond traditional keyword matching.

Instead of selecting candidates only because their resumes contain specific words, TalentLens AI evaluates candidate relevance using intelligent ranking signals and candidate profiling.

---

## Problem

Traditional hiring systems often miss strong candidates because they depend heavily on keyword matching.

TalentLens AI focuses on identifying high-potential candidates by combining:

* Candidate profile understanding
* Experience evaluation
* AI-oriented relevance scoring
* Explainable ranking
* Hidden skill discovery

---

## Features

### AI Candidate Ranking

Ranks candidates using profile signals instead of direct keyword counting.

### Explainable AI

Each shortlisted candidate includes:

* Why Ranked
* Overall fit indicators
* Candidate reasoning signals

### Hidden Skill Discovery

Detects likely skills from candidate profiles and surfaces them for recruiters.

Example:

Headline:
AI Specialist | Building ML-powered solutions

Detected:

* AI
* Machine Learning
* LLM

### Fast Shortlisting Engine

Processes candidate data efficiently and returns ranked candidates.

### Interactive Dashboard

Modern React interface displaying:

* Top candidates
* Scores
* Experience
* AI explanations
* Detected skills
